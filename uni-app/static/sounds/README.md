# 抽卡音效文件

请将以下音效文件放入此目录：

## 需要的音效文件

| 文件名 | 用途 | 建议时长 |
|--------|------|----------|
| `gacha_common.mp3` | 普通品质抽卡音效 | 1-2秒 |
| `gacha_rare.mp3` | 稀有品质抽卡音效 | 2-3秒 |
| `gacha_legendary.mp3` | 传说品质抽卡音效 | 3-4秒 |

## 音效风格建议

- **gacha_common**: 清脆的"叮"声，简单明快
- **gacha_rare**: 更丰富的和弦，带有惊喜感
- **gacha_legendary**: 震撼的管弦乐，带有史诗感

## 免费音效资源

- https://freesound.org/
- https://mixkit.co/free-sound-effects/game/
- https://www.sonnybox.com/

## 注意事项

- 格式：MP3（推荐）或 WAV
- 采样率：44100Hz
- 声道：单声道或立体声均可
- 文件大小：建议不超过 500KB
